OPTI-G

This is an optimization program for G code, and this is done by reorder the cuts, to minimize the distance between those. Use it entirely at your own risk! It can still contain bugs!

The program:

There is no different between how the software handle Etch and Drill files, and other G-code files can also be optimized by this program.

The optimization:

The optimizing support both where the cut enter and stop at different places and the cut enter and exit the same place. The used optimizing do NOT support G-code, where the order of the cuts between tool change are important, and need to be performed in sequence. If you are interested in the algorithm used to optimize the G-code more information can be found on this link.

Installation:

  1. Download the project and unpack it.
  2. Now there are two options
    • Move the 'Program' folder to location you want, and run the program from here. Make sure you have .NET 4.5 to run the program.
    • Run the setup.exe file from the 'Installation files' folder.
